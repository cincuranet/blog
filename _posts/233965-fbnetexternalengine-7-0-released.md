Step forward for [_FbNetExternalEngine_][1] today - version 7.0 is ready, you can go and download it right now.

<!-- excerpt -->

This version brings update of runtime to .NET 10, so you can use all the latest and greatest features and also benefit from [numerous performance improvements there][3]. On the _FbNetExternalEngine_ itself I see about 3% speed performance.

Every user of 6.x version gets free upgrade to 7.x.

As usual, check the [docs page][1] as well as the _examples_ directory in the [downloaded package][2].
